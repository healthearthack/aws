Usage Example — StatCard Component

// packages/ui/src/components/StatCard.tsx
import { StatCard } from 'magic2u-design-system/ui/react';

export function MetricsPanel() {
  const { data } = useMetrics();
  return (
    <div className="grid grid-cols-2 gap-md">
      <StatCard
        label="Members"
        value={data.members.toLocaleString()}
        delta="+12%"
        trend="up"
      />
    </div>
  );
}

Component Spec — StatCard

The StatCard component renders a metric with a label, value, optional delta, and trend indicator.

labelstring — Display text
valuestring — Formatted metric value
delta?string — Change indicator (+12%)
trend?"up" | "down" | "neutral"
